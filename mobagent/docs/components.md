# Components

| Component | Purpose |
|-----------|---------|
| `AgentCard` | Home agent summary card (progress, tokens, cost) |
| `TimelineItem` | Structured event row with icon |
| `ConsoleView` | Raw log stream, search, pause scroll |
| `DiffList` | Changed files list (+/- counts) |
| `DiffViewer` | Inline diff with add/remove colors |
| `AppNav` | Bottom tab bar (Telegram one-hand) |

Pages: `HomePage`, `AgentDetailPage` (tabs), `NewTaskPage`, `SettingsPage`.

Animations: 200ms transitions on cards/tabs; progress bar width transition 300ms.
